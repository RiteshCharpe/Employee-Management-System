# Employee-Management-System

Full-stack Employee Management System using Spring Boot + React

A full-stack Employee Management System built with **Spring Boot (Backend)** and **React (Frontend)**.  
This project demonstrates CRUD operations, RESTful APIs, and seamless integration between backend and frontend.

---

## 🚀 Features
- Add, update, delete, and view employees
- REST API with Spring Boot
- Responsive frontend with React
- MySQL database integration
- Clear separation of backend and frontend layers

---

## 🛠️ Tech Stack
- **Backend:** Spring Boot, Maven, Lombok, JPA/Hibernate
- **Frontend:** React, Axios, Bootstrap
- **Database:** MySQL
- **Tools:** IntelliJ IDEA / VS Code, Git, GitHub
